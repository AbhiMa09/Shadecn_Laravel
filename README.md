# Shadecn Laravel UI

A port of the [shadcn/ui](https://ui.shadcn.com/) components for Laravel Blade, styled with Tailwind CSS and powered by Alpine.js.

## Stack

- **Laravel 12.x**
- **Tailwind CSS 4**
- **Alpine.js**
- **Vite**

## Installation

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
```

## Local Development

```bash
npm run dev
php artisan serve
```

## Docker

```bash
docker-compose up -d
```
